# ЕГЭ История — Telegram-бот

Бот для подготовки к ЕГЭ по Истории России. Вопросы организованы по кодификатору ФИПИ.

## Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Создание бота

1. Открой [@BotFather](https://t.me/BotFather) в Telegram
2. Напиши `/newbot` и следуй инструкциям
3. Скопируй токен

### 3. Настройка

```bash
cp .env.example .env
# Открой .env и вставь токен:
# BOT_TOKEN=1234567890:AAAA...
```

### 4. Парсинг вопросов

```bash
python parser.py
```

Парсер соберёт до 50 вопросов на каждую тему кодификатора с сайта решу.егэ.  
Можно изменить лимит: `run_parser(max_per_topic=100)` в конце файла.

### 5. Запуск бота

```bash
python bot.py
```

## Структура проекта

```
ege_history_bot/
├── bot.py          # точка входа
├── parser.py       # парсер вопросов с sdamgia.ru
├── database.py     # работа с SQLite
├── config.py       # настройки
├── handlers/
│   ├── start.py    # /start, главное меню
│   ├── topics.py   # выбор раздела и темы
│   └── quiz.py     # логика квиза
└── requirements.txt
```

## Функциональность

- 📋 Выбор темы из кодификатора (12 разделов, 50+ тем)
- 🎯 Вопросы с вариантами ответов и открытые задания
- ✅ Проверка ответа + объяснение
- 📊 Личная статистика (решено / правильных)
- 🔁 Случайный порядок вопросов без повторений
